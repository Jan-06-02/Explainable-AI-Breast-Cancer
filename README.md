# Explainable AI for Breast Cancer Prediction

## Project Overview
This project predicts breast cancer diagnosis using a Random Forest Classifier and explains model predictions using SHAP and LIME.

## Features
- Data Preprocessing
- Random Forest Classification
- Model Evaluation
- SHAP Explanations
- LIME Explanations
- Explainable AI Visualizations

## Technologies
- Python
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib
- SHAP
- LIME

## Dataset
Breast Cancer Dataset
